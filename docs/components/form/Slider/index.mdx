# Slider 滑块

滑块组件允许用户在一个范围内选择一个数值。

## 功能特性

- 支持单滑块数值选择
- 可配置最小值、最大值和步长
- 支持自定义刻度标记
- 可切换水平/垂直方向
- 支持正向/反向显示
- 提供工具提示
- 支持多种尺寸和变体样式
- 支持受控/非受控模式
- 提供丰富的事件和Ref方法

## 基本用法

```jsx
import { useState } from 'react';
import { Slider } from '@taro-uno/components';

export default function SliderDemo() {
  const [value, setValue] = useState(50);

  return (
    <Slider value={value} onChange={setValue} />
  );
}
```

## 使用示例

### 基础滑块

```jsx
<Slider defaultValue={30} />
```

### 受控模式

```jsx
import { useState } from 'react';
import { Slider } from '@taro-uno/components';

export default function SliderDemo() {
  const [value, setValue] = useState(50);

  return (
    <Slider value={value} onChange={setValue} />
  );
}
```

### 范围和步长

```jsx
<Slider min={0} max={200} step={10} defaultValue={100} />
```

### 刻度标记

```jsx
<Slider
  min={0}
  max={100}
  marks={{
    0: '0°C',
    25: '25°C',
    50: '50°C',
    75: '75°C',
    100: '100°C'
  }}
  defaultValue={50}
/>
```

### 垂直方向

```jsx
<Slider vertical defaultValue={30} style={{ height: 200 }} />
```

### 反向显示

```jsx
<Slider reverse defaultValue={30} />
```

### 禁用状态

```jsx
<Slider disabled defaultValue={30} />
```

### 自定义工具提示

```jsx
<Slider
  tooltip={{
    formatter: (value) => `${value}%`
  }}
  defaultValue={30}
/>
```

### 不显示工具提示

```jsx
<Slider tooltip={false} defaultValue={30} />
```

### 不同尺寸

```jsx
<Slider size="small" defaultValue={30} />
<Slider size="medium" defaultValue={30} />
<Slider size="large" defaultValue={30} />
```

## API

### SliderProps

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| min | `number` | `0` | 最小值 |
| max | `number` | `100` | 最大值 |
| step | `number` | `1` | 步长 |
| value | `number` | - | 当前值，受控模式 |
| defaultValue | `number` | `0` | 默认值，非受控模式 |
| size | `'small' | 'medium' | 'large'` | `'medium'` | 滑块尺寸 |
| variant | `'default' | 'filled' | 'outlined'` | `'default'` | 滑块变体 |
| disabled | `boolean` | `false` | 是否禁用 |
| marks | `Record<number, React.ReactNode>` | - | 刻度标记 |
| included | `boolean` | `true` | 是否包含已选择部分 |
| vertical | `boolean` | `false` | 是否垂直方向 |
| reverse | `boolean` | `false` | 是否反向 |
| tooltip | `{ visible?: boolean; formatter?: (value: number) => React.ReactNode; } | boolean` | - | 工具提示配置 |
| className | `string` | - | 自定义样式类名 |
| style | `React.CSSProperties` | - | 自定义样式 |
| onChange | `(value: number) => void` | - | 值变化时的回调 |
| onChangeComplete | `(value: number) => void` | - | 拖拽结束时的回调 |
| accessible | `boolean` | - | 是否启用无障碍访问 |
| accessibilityLabel | `string` | - | 无障碍标签 |
| accessibilityRole | `string` | - | 无障碍角色 |
| accessibilityState | `{ disabled?: boolean; readonly?: boolean; busy?: boolean; }` | - | 无障碍状态 |

### SliderRef

| 方法 | 类型 | 说明 |
| --- | --- | --- |
| getValue | `() => number` | 获取当前值 |
| setValue | `(value: number) => void` | 设置值 |
| disable | `() => void` | 禁用滑块 |
| enable | `() => void` | 启用滑块 |

### 类型定义

```typescript
export type SliderSize = 'small' | 'medium' | 'large';
export type SliderVariant = 'default' | 'primary' | 'secondary' | 'success' | 'warning' | 'error';
```

## 使用提示

1. **受控与非受控模式**：通过设置`value`和`onChange`可以使用受控模式，否则为非受控模式。
2. **刻度标记**：`marks`属性可以自定义刻度，键为数值，值为显示的标签。
3. **垂直滑块**：设置`vertical`为`true`并指定高度即可使用垂直滑块。
4. **工具提示**：可以通过`tooltip`属性自定义工具提示的显示和格式。
5. **禁用状态**：通过`disabled`属性可以禁用滑块，此时无法进行交互。

## 浏览器兼容性

- 微信小程序
- 支付宝小程序
- 百度小程序
- 字节跳动小程序
- H5

## 无障碍支持

- 支持`role="slider"`属性
- 提供`aria-valuemin`、`aria-valuemax`和`aria-valuenow`属性
- 支持`aria-disabled`状态
- 可自定义`accessibilityLabel`和`accessibilityRole`

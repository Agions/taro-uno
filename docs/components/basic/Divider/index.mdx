# Divider 分割线

Divider 是用于在内容中创建视觉分隔的组件，可以是水平或垂直方向，支持多种样式和配置。

## 功能介绍

- 支持水平和垂直两种方向
- 提供多种线条样式（实线、虚线、点线等）
- 可添加文本或图标作为分割内容
- 支持响应式设计
- 支持渐变效果和动画
- 提供无障碍访问支持

## 使用示例

### 默认分割线

```tsx
import { Divider } from '@taro-uno/ui';

function BasicDivider() {
  return <Divider />;
}
```

### 方向设置

```tsx
import { Divider } from '@taro-uno/ui';

function DirectionDivider() {
  return (
    <>
      <Divider orientation="horizontal" />
      <div style={{ display: 'flex', alignItems: 'center' }}>
        <span>左侧内容</span>
        <Divider orientation="vertical" style={{ height: '20px', margin: '0 16px' }} />
        <span>右侧内容</span>
      </div>
    </>
  );
}
```

### 线条类型

```tsx
import { Divider } from '@taro-uno/ui';

function TypeDivider() {
  return (
    <>
      <Divider type="solid" />
      <Divider type="dashed" />
      <Divider type="dotted" />
      <Divider type="double" />
    </>
  );
}
```

### 带文本的分割线

```tsx
import { Divider } from '@taro-uno/ui';

function TextDivider() {
  return (
    <>
      <Divider>中间文本</Divider>
      <Divider position="left">左侧文本</Divider>
      <Divider position="right">右侧文本</Divider>
    </>
  );
}
```

### 带图标和文本的分割线

```tsx
import { Divider, Icon } from '@taro-uno/ui';

function IconTextDivider() {
  return (
    <Divider>
      <Icon name="heart" color="#ff4d4f" />
      <span style={{ marginLeft: '8px' }}>喜欢</span>
    </Divider>
  );
}
```

### 颜色和尺寸

```tsx
import { Divider } from '@taro-uno/ui';

function ColorSizeDivider() {
  return (
    <>
      <Divider color="primary" size="xs" />
      <Divider color="success" size="sm" />
      <Divider color="warning" size="md" />
      <Divider color="error" size="lg" />
      <Divider color="info" size="xl" />
    </>
  );
}
```

### 渐变分割线

```tsx
import { Divider } from '@taro-uno/ui';

function GradientDivider() {
  return (
    <Divider
      gradient={{
        start: '#1890ff',
        end: '#52c41a',
        direction: 'to right'
      }}
      height={2}
    />
  );
}
```

### 动画分割线

```tsx
import { Divider } from '@taro-uno/ui';

function AnimatedDivider() {
  return (
    <Divider
      animated
      animationDuration={1000}
      type="dashed"
      color="primary"
    />
  );
}
```

### 可点击的分割线

```tsx
import { Divider } from '@taro-uno/ui';

function ClickableDivider() {
  const handleClick = () => {
    console.log('Divider clicked');
  };

  return (
    <Divider
      clickable
      onClick={handleClick}
      children="点击我"
      style={{ cursor: 'pointer' }}
    />
  );
}
```

### 响应式分割线

```tsx
import { Divider } from '@taro-uno/ui';

function ResponsiveDivider() {
  return (
    <Divider
      responsive
      breakpoint="md"
      orientation="horizontal"
      children="响应式分割线"
    />
  );
}
```

## API

### DividerProps

| 属性名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| orientation | DividerOrientation | 'horizontal' | 分割线方向，可选值：'horizontal'、'vertical' |
| type | DividerType | 'solid' | 分割线类型，可选值：'solid'、'dashed'、'dotted'、'double'、'groove'、'ridge'、'inset'、'outset' |
| position | DividerPosition | 'center' | 分割线位置，可选值：'left'、'center'、'right' |
| size | DividerSize | 'md' | 分割线尺寸，可选值：'xs'、'sm'、'md'、'lg'、'xl' |
| color | DividerColor | 'border' | 分割线颜色，可选值：'primary'、'secondary'、'success'、'warning'、'error'、'info'、'light'、'dark'、'border' 或自定义颜色字符串 |
| variant | DividerVariant | 'default' | 分割线变体，可选值：'default'、'inset'、'middle'、'text'、'with-icon' |
| children | ReactNode | - | 分割线文本内容 |
| inset | boolean | false | 是否边距内缩 |
| centered | boolean | false | 是否居中显示 |
| className | string | - | 自定义样式类名 |
| style | React.CSSProperties | - | 自定义样式 |
| width | number \| string | - | 分割线宽度 |
| height | number \| string | - | 分割线高度 |
| margin | number \| string | - | 分割线边距 |
| padding | number \| string | - | 分割线内边距 |
| opacity | number | 1 | 分割线透明度 |
| borderRadius | number \| string | 0 | 分割线圆角 |
| gradient | { start: string; end: string; direction?: string } | - | 分割线渐变配置 |
| animated | boolean | false | 是否启用动画 |
| animationDuration | number | 300 | 动画持续时间（毫秒） |
| shadow | boolean | false | 是否显示阴影 |
| clickable | boolean | false | 是否可点击 |
| onClick | CommonEventFunction<any> | - | 点击事件处理函数 |
| accessible | boolean | true | 是否启用无障碍访问 |
| accessibilityLabel | string | - | 无障碍标签 |
| accessibilityRole | string | 'separator' | 无障碍角色 |
| spacing | number \| string | - | 分割线间距 |
| align | 'start' \| 'center' \| 'end' \| 'stretch' | - | 分割线对齐方式 |
| verticalAlign | 'top' \| 'middle' \| 'bottom' | - | 分割线垂直对齐方式 |
| icon | ReactNode | - | 分割线图标 |
| iconPosition | 'start' \| 'center' \| 'end' | 'center' | 图标位置 |
| responsive | boolean | false | 是否响应式 |
| breakpoint | 'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' | 'md' | 响应式断点 |
| textStyle | React.CSSProperties | - | 文本样式 |
| textSpacing | number \| string | 16 | 文本间距 |
| textBackground | string | '#ffffff' | 文本背景色 |
| textPadding | number \| string | '0 16px' | 文本内边距 |
| textBorderRadius | number \| string | 4 | 文本圆角 |

| 方法名 | 类型 | 说明 |
|--------|------|------|
| element | HTMLDivElement \| null | 分割线元素 |

| getColor | () => string | 获取分割线颜色 |
| setOrientation | (orientation: DividerOrientation) => void | 设置分割线方向 |
| setType | (type: DividerType) => void | 设置分割线类型 |
| setPosition | (position: DividerPosition) => void | 设置分割线位置 |
| setSize | (size: DividerSize) => void | 设置分割线尺寸 |
| setColor | (color: string) => void | 设置分割线颜色 |
| scrollIntoView | (options?: ScrollIntoViewOptions) => void | 滚动到视图 |

## 使用提示

1. 水平分割线默认宽度为100%，垂直分割线默认高度为100%
2. 当同时设置children和icon时，会自动切换到'with-icon'变体
3. 渐变效果会覆盖原有的边框颜色设置
4. 点击事件仅在clickable为true时生效
5. 响应式分割线会根据屏幕尺寸自动调整方向
6. 文本分割线的背景色默认为白色，确保文本在不同背景下可见

## 无障碍支持

- 默认设置了role="separator"，确保屏幕阅读器能正确识别
- 可通过accessibilityLabel属性自定义无障碍标签
- 支持通过accessibilityRole属性自定义无障碍角色

## 浏览器兼容性

支持所有现代浏览器以及Taro支持的小程序平台。
